# Cadastro de Roupas

Sistema simples de cadastro de produtos desenvolvido com foco em **usabilidade (UX)** e aplicação das **Heurísticas de Nielsen**.

---

## Sobre o projeto

Este projeto consiste em um formulário para cadastro de produtos de uma **loja de roupas**, permitindo o registro de informações como:

- Nome do produto
- Categoria
- Gênero
- Tamanho
- Cor
- Preço
- Quantidade em estoque
- Descrição

O objetivo principal é aplicar conceitos de **Interação Humano-Computador (IHC)**, garantindo uma interface intuitiva, clara e eficiente para o usuário.

---

## Objetivo acadêmico

Este projeto foi desenvolvido como atividade prática para aplicar as **Heurísticas de Nielsen**, com foco em:

- Visibilidade do estado do sistema  
- Prevenção de erros  
- Consistência e padrões  

---

## Funcionalidades

✔ Validação em tempo real dos campos  
✔ Máscara automática para o campo de preço (R$)  
✔ Feedback visual para erros e sucesso  
✔ Interface responsiva e organizada  
✔ Campos obrigatórios destacados  

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
